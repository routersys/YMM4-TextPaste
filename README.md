## 概要

本プラグインは YMM4 の映像エフェクトとして動作します。
任意のテキストを指定した位置・サイズ・回転・不透明度で描画し、元アイテムに対して4種類の合成モードで重ね合わせることができます。

処理は YMM4 内部の `TextItem` / `ISource` を用いてテキストを GPU 上でレンダリングし、`ColorMatrix` で不透明度を制御したのち、`AffineTransform2D` で位置・拡大率・回転・Z 軸遠近を適用し、最終的に `Composite` エフェクトで元映像と合成する、という形で行われます。

## インストール方法

1. [Releases](https://github.com/routersys/YMM4-TextPaste/releases) ページから最新のプラグインファイル（`.ymme`）をダウンロードしてください。
2. YMM4 が起動していないことを確認し、ダウンロードしたファイルを実行（ダブルクリック）してインストールします。
3. YMM4 を起動し、映像エフェクトの追加から **「テキストの貼り付け」** を選択してください。

## 機能

### 1. テキスト描画

任意のテキストを YMM4 の `TextItem` を介して GPU 上でレンダリングし、コマンドリストとして記録します。テキストにはリッチテキスト形式の装飾、フォント、文字色、文字サイズ、文字間隔、行の高さを個別に設定できます。

- フレームごとにコマンドリストを再構築し、アニメーション値の変化に追従します。
- テキストが空であっても処理はスキップされず、透明な出力として扱われます。

### 2. 変形・配置

描画されたテキストに対して、以下の変換を `ColorMatrix` および `AffineTransform2D` エフェクトで適用します。

- **不透明度**: `ColorMatrix` のアルファ成分（M44）を乗算することで不透明度を制御します。
- **X / Y 移動**: `AffineTransform2D` の平行移動行列で描画位置を移動します。
- **Z 座標（遠近）**: 透視投影（視点距離 1000）を用いたスケール係数を算出し、`Zoom` と乗算します。分母がゼロ近傍になる場合は最小スケール（1e-5）でクランプします。
- **拡大率（Zoom）**: `AffineTransform2D` のスケール変換で適用します。Z 座標の遠近スケールと合算されます。
- **回転**: 度数法の値をラジアンに変換し、`AffineTransform2D` の回転変換で適用します。
- **変換順序**: スケール → 回転 → 平行移動の順に行列を合成します。

### 3. 表示方法の切り替え

`Composite` エフェクトの合成モードを切り替えることで、元アイテムとテキストの合成方法を変更できます。詳細は[表示方法の詳細](#表示方法の詳細)を参照してください。

### 4. 多言語対応

ローカライズファイルを内蔵しており、YMM4 の言語設定に合わせてエフェクト名およびパラメータ名が自動で切り替わります。

対応言語：日本語 (`ja-jp`)、英語 (`en-us`)、中国語簡体字 (`zh-cn`)、中国語繁体字 (`zh-tw`)、韓国語 (`ko-kr`)、スペイン語 (`es-es`)、アラビア語 (`ar-sa`)、インドネシア語 (`id-id`)

## パラメータ一覧

### 描画グループ

| パラメータ | 種別 | 既定値 | 範囲 | 単位 | 説明 |
|---|---|---|---|---|---|
| X | アニメーション | 0 | ±無制限 | px | 描画位置の X 座標 |
| Y | アニメーション | 0 | ±無制限 | px | 描画位置の Y 座標 |
| Z | アニメーション | 0 | ±無制限 | px | 描画位置の Z 座標（遠近投影に使用） |
| 不透明度 | アニメーション | 100 | 0 〜 100 | % | 描画の不透明度 |
| 拡大率 | アニメーション | 100 | 0 〜 無制限 | % | 描画の拡大率 |
| 回転角 | アニメーション | 0 | ±無制限 | ° | 描画の回転角 |

### 表示グループ

| パラメータ | 種別 | 既定値 | 説明 |
|---|---|---|---|
| 表示方法 | コンボボックス | そのまま重ねて表示 | テキストの表示方法（合成モード）を選択します |
| 文字揃え位置 | コンボボックス | 中央中央 | テキストの揃え位置を設定します |

### テキストグループ

| パラメータ | 種別 | 既定値 | 範囲 | 単位 | 説明 |
|---|---|---|---|---|---|
| テキスト | リッチテキスト | （空） | — | — | 貼り付けるテキスト（装飾・色の個別指定が可能） |
| フォント | コンボボックス | メイリオ | — | — | テキストのフォント |
| サイズ | アニメーション | 34 | 1 〜 無制限 | px | 文字の大きさ |
| 文字色 | カラーピッカー | 白 | — | — | 文字の色 |
| 文字間隔 | アニメーション | 0 | ±無制限 | px | 文字と文字の間隔 |
| 行の高さ | アニメーション | 100 | 0 〜 無制限 | % | 行の高さの割合 |

## 表示方法の詳細

| 値 | 合成モード | 説明 |
|---|---|---|
| 完全置換 | `SourceOver`（背景: 透明） | 元アイテムを消してテキストのみを表示します |
| 領域内に表示 | `SourceIn` | 元アイテムの領域内にのみテキストを表示します |
| 領域上に重ねて表示 | `SourceAtop` | 元アイテムを維持し、その上の領域にテキストを表示します |
| そのまま重ねて表示 | `SourceOver`（背景: 元アイテム） | 元アイテムを維持したまま、その上にテキストを重ねて表示します |

## 注意事項

- AviUtl フィルタ形式（`.exo` 出力）には対応していません。
- 本エフェクトはフレームごとにテキストのコマンドリストを再構築するため、テキストが長いほど処理負荷が増加します。
- `Z` 座標を元アイテムの視点距離（1000）以上に設定した場合、映像が反転・消失する場合があります。
- キーフレームによるアニメーションはすべてのパラメータ（X、Y、Z、不透明度、拡大率、回転角、サイズ、文字間隔、行の高さ）で対応しています。
